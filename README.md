# Bloom Valley Nursery

A client showcase website for Bloom Valley Nursery - a popup shop that sells seeds, flowers, plants, and fruits at local farmers markets. The site serves as an online presence to showcase their products, share their story, and connect with customers between market days. The original site is no longer active. The link below showcases the front-end design only, as the backend is no longer running.

View the website here: [ https://ashley062.github.io/bloomvalleynurseryproject/shopGallery.html ]

## Pages

- **Home** — hero section, best sellers, and customer testimonials
- **About Us** — the nursery's story and mission
- **Shop Gallery** — full product catalog with add-to-cart functionality
- **PlantFinder Quiz** — an interactive quiz to help customers find the right plant for their space

## Features

- Responsive design with a mobile hamburger menu
- Add to cart with live cart item count in the nav
- PlantFinder interactive quiz
- Email newsletter subscription form
- Customer testimonials section
- Seasonal sale promotions
- Social media links
- SEO meta description

## Tech Stack

- **HTML** — page structure and content
- **CSS** — custom styling and responsive layout
- **JavaScript** — cart functionality, quiz logic, and interactivity
- **Font Awesome** — icons
- **Google Fonts** — typography
