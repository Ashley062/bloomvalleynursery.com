# Bloom Valley Nursery

A client showcase website for Bloom Valley Nursery — a popup shop that sells seeds, flowers, plants, and fruits at local farmers markets. The site serves as an online presence to showcase their products, share their story, and connect with customers between market days.

## Pages

- **Home** — hero section, best sellers, and customer testimonials
- **About Us** — the nursery's story and mission
- **Shop Gallery** — full product catalog with add-to-cart functionality
- **PlantFinder Quiz** — an interactive quiz to help customers find the right plant for their space

## Features

- Responsive design with a mobile hamburger menu
- Add to cart with live cart item count in the nav
- PlantFinder interactive quiz
- Email newsletter subscription form
- Customer testimonials section
- Seasonal sale promotions
- Social media links
- SEO meta description

## Tech Stack

- **HTML** — page structure and content
- **CSS** — custom styling and responsive layout
- **JavaScript** — cart functionality, quiz logic, and interactivity
- **Font Awesome** — icons
- **Google Fonts** — typography

## Deployment

Since this is a static site (HTML/CSS/JS only), it's easy and free to deploy. Here are the best options:

### GitHub Pages (Recommended — free & easy)
Since the project is already on GitHub:
1. Go to the repo on GitHub
2. Click **Settings** → **Pages**
3. Under **Branch**, select `main` and click **Save**
4. Your site will be live at `https://ashley062.github.io/bloomvalleynurseryproject/`

### Netlify (Free — drag & drop)
1. Go to [netlify.com](https://netlify.com) and sign in with GitHub
2. Click **Add new site** → **Import an existing project**
3. Select this repo — Netlify auto-deploys on every push
4. You get a free custom URL like `https://bloom-valley-nursery.netlify.app`

### Vercel (Free — also great for static sites)
1. Go to [vercel.com](https://vercel.com) and sign in with GitHub
2. Click **Add New Project** and import this repo
3. Vercel auto-deploys and gives you a public URL instantly

All three options auto-redeploy whenever you push changes to GitHub. **GitHub Pages** requires no additional account setup since you're already using GitHub.
